# YUNFEI Auto Compress

YUNFEI Auto Compress is a macOS utility that monitors folders and automatically compresses matching videos in batches.

## Key Features
- Auto-monitor folders and queue new videos by keyword
- Batch compression with H.264/H.265
- Output modes: overwrite, output folder, or suffix
- Queue progress, logs, and status visibility
- Auto-detect and download ffmpeg
- Single-window behavior

## Requirements
- macOS 13.0+
- ffmpeg (auto-download supported)
